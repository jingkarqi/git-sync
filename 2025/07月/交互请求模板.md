# 5. 交互请求模板（Agent 向用户发起）  
请使用以下固定句式，不得自行改写：  

1. **请求读取文件**  
```
    请上传文件：<文件名>，以便我继续下一步分析。
```

2. **请求代码修改**  
	***请求修改如下代码：***
	- **原始代码**
	```matlab
	sales_table = sales_table(strcmp(sales_table.销售类型, '销售'), :);
	category_daily_sales = groupsummary(merged_data, {'销售日期', '分类名称'}, 'sum', '销量_千克_');
	```
    
	- **修改后代码**
	```matlab
	% 通过detectImportOptions指定英文列名
	opts_sales = detectImportOptions('附件2.xlsx');
	opts_sales.VariableNames = {'sale_date', 'scan_time', 'item_code', 'sale_kg', 'unit_price', 'sale_type', 'is_discount'};
	sales_table = readtable('附件2.xlsx', opts_sales);
	
	% 使用英文变量名筛选和聚合
	sales_table = sales_table(strcmp(sales_table.sale_type, '销售'), :);
	category_daily_sales = groupsummary(merged_data, {'sale_date', 'category_name'}, 'sum', 'sale_kg');
	```
