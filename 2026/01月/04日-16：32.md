# 管理科学与工程学科是什么？

1. **学科定义与性质**：管理科学与工程是一级学科。它是综合运用系统科学、管理科学、数学、经济和行为科学及工程方法，结合信息技术研究解决社会、经济、工程等方面的管理问题的一门学科。
2. **学科特点**：该学科是管理理论与管理实践紧密结合的学科，侧重于研究同现代生产、经营、科技、经济、社会等发展相适应的管理理论、方法和工具。
3. **培养目标**：该学科培养学生具有扎实系统的管理理论基础，合理的知识结构，并能正确地应用系统分析方法及相应的工程技术方法解决管理方面的有关理论与实际问题。
4. **学科分类**：在国家的学科分类中，它属于“管理学”门类（代码12）下的一级学科，代码为1201，并可授予管理学或工学学位。

# 2. 管理科学理论（规律）的一般表达结构是什么？

1. **什么条件**（环境、前提、规则、规定、观念）。
2. **什么群体**（国家、阶级、阶层、民族、组织、团体、机构、企业、单位、某类人群等）。
3. **基于什么物质条件**（设备、工具）。
4. **什么行为**（或某类行为），（或者，将、已发生的什么行为）。
5. **会（将，已）导致什么结果**。
6. **如何控制、引导、管理，或者如何调整这些行为，才能得到更有利（于什么群体的人的）的结果**（包括优化问题）。

文档进一步概括为两个方面：**行为（因果和演变）规律**；**得到更好结果（优化）的方法**。

# 3. 管理科学与工程研究的对象是什么？

1. **含人的系统**：即研究对象直接包含人（个体或群体）。
2. **不含人的系统**：具体指由人的行为结果所构成的系统，例如金融市场、财务系统、技术系统等。文档同时也将“自然系统”列为研究对象。

此外，文档在概述部分指出，该学科是研究解决**社会、经济、工程等方面的管理问题**。在论文写作部分举例说明，研究对象可以是：企业、流域、国家公园、业务流程、生态系统、政策等。

# 4. 管理科学所要研究的问题是什么？
根据文档《第二讲 管理科学研究范式.pptx》中“2.5管理科学所要研究的问题是：”部分，管理科学所要研究的问题具体包括以下三个方面：

1. **新时空中，研究对象（特征、变量，新的延伸）的现状（陈述、指标、均值、方差、指标分量的综合）**。
2. **新时空中，变量间的关系，及变量的演变规律**。
3. **基于前两点，产生的管理对策（政策、措施、优化方法和模型）**。
