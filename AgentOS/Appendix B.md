# Appendix B: Seed Vocabulary

To avoid conceptual drift, the following words should remain sharply defined:

- Principal
- Task
- State
- Event
- Capability
- Checkpoint
- Resume
- Policy Authority
- Evidence
- Memory
- Projection
- Budget
- Control Signal
- Handoff
- Trust Zone
- Operator
- Tenant

The health of the system depends on vocabulary discipline.
